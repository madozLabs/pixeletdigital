# Pixel&Digital — Visual Foundations

Status: Provisional working foundation; owner approval required
References: `PIXEL_DIGITAL_BRAND_BIBLE.md`, `PIXEL_DIGITAL_LOGO_REGISTER.md`

## 1. Evidence boundary

Observed: four 1418 × 1418 PNG source assets are versioned in the repository. Two use transparency and two use an opaque red gradient field. The symbol raster contains an observed red near `#E81B2C`.

Not approved: universal colour tokens, gradient stops, typography, spacing rules, minimum sizes, clear space, print values, vector construction or usage rights.

## 2. Provisional working rules

- Use the existing raster assets only as traceable review and layout inputs.
- Preserve their proportions; never redraw, stretch, recolour or reconstruct the gradient.
- Use the transparent symbol or lockup only where raster resolution is sufficient.
- Keep layouts editorial, precise and restrained; red supports hierarchy but must not dominate every surface.
- Use neutral working surfaces when final supporting colours are still undecided.
- Typography exploration must use clearly labelled candidates; no candidate may be called the official wordmark font.

## 3. Accessibility and contrast

- Validate every foreground/background pairing; the observed red alone does not guarantee accessible contrast.
- Never communicate status, selection or meaning through red alone.
- Preserve visible focus, readable labels and complete keyboard paths.
- All motion, sound and 3D treatments require an equivalent static and reduced-motion experience.

## 4. Mobile and performance

- Prioritize promise, proof and primary action before decorative media.
- Prefer the symbol where the full lockup loses legibility, subject to later minimum-size approval.
- Do not make a hero dependent on a large raster, autoplay video or 3D scene.
- Reserve media dimensions and provide lightweight fallbacks.

## 5. Exploration lanes

Permitted for review: controlled red-led compositions, black/white/neutral contrast studies, editorial grids, kinetic type storyboards, authentic photography and restrained depth.

Prohibited without owner approval: final tokens, invented gradients, font claims, logo reconstruction, decorative 3D, aggressive motion, unsupported claims or Studio/Training full-world visuals.

## 6. Approval gate

Final visual standards require owner-approved vector/master files, rights authority, logo usage rules, typography, colour system, gradient specification, clear space, minimum size and tested contrast combinations.
