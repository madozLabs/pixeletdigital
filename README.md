# Pixel&Digital

Minimal full-stack application scaffold for the Pixel&Digital platform.

## Local development

Prerequisites: a current Node.js release and npm.

```sh
npm install
npm run dev
```

Open `http://localhost:3000`.

Run all repository checks with:

```sh
npm run check
npm run build
```
